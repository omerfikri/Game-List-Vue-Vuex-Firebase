# Game-List-Vue-Vuex-Firebase
The project is a simple website. But it includes vuex state management. The user logs in anonymus to website. The informations stores in firebase.

# Requirements
  - firebase : ^10.8.0,
  - gsap :  ^3.12.5 ,
  - vue :  ^3.4.15,
  - vue-router: ^4.2.5,
  - vuex: ^4.0.2
